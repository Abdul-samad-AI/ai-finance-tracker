# AI Finance Tracker

## Description
AI Finance Tracker is a full-stack web application that helps users track expenses, analyze spending patterns, and gain financial insights using AI-based analysis.

## Tech Stack
- Frontend: React
- Backend: Django, Django REST Framework
- Database: PostgreSQL
- Authentication: JWT (planned)
- AI: Basic ML models (planned)

## Features
- Expense tracking
- Category-wise financial analysis
- Monthly and yearly reports
- AI-powered spending insights (future scope)
